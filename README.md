## Список технологий проекта

* ### ```Лаунчер```
  - ### Java Native Access (JNA)
    > Используется для доступа к нативным функциям ОС, таким как WinAPI (user32.dll)
    
  - ### HiDPI в Swing 
    > Обычные UI элементы, встроенные в Swing корректно не поддерживают повышенный масштаб интерфейса. 
    > Это вызывает эффект мыльности на мониторах с высоким разрешением. 
    > Для решения этой проблемы было переписано большое количество элементов UI с собственной отрисовкой
    
  - ## Размытие по гауссу
    > Дизайн UI требовал размытие заднего фона, из-за чего и было написано несколько алгоритмов размытия по гауссу.
    > Присутствует как обычный алгоритм, так и оптимизированный (Box blur). Позже, в целях чистоты кода, была использована
    > [библиотека с готовым решением](http://www.jhlabs.com/ip/blurring.html)
    
  - ## WinAPI
    > Используется для изменения иконки и заголовка окна игры, а так же для получения Id процесса на Windows.
    > Ещё имеется возможность получить текущую тему системы (тёмная или светлая), но применялась только для тестов
  
  - ## JOGL (OpenGL)
    > Для отображения скина игрока в профиле создаётся 3D моделька. Она имеет анимации, а так же возможность ношения плаща.
    > Исходный код был адаптирован для HiDPI

  - ## Парсинг
      * ### YouTube
        > Доступна возможность узнавать количество подписчиков, а так же получать иконку канала.
        > Так же можно получать список последних видео с заголовком и превью в высоком разрешении
      * ### VK
        > Имеется возможность получения описания, иконки, а так же количества участников сообщества.
        > Вместе с этим можно получать последние записи с разными типами - Текст, Фото, Видео, Сниппет
      * ### Instagram
        > Обычное получение информации о последних фотографиях пользователя, используя небольшую лазейку в API инстаграма

  - ## Discord IPC
    > Отображение информации об игроках сервера Minecraft
